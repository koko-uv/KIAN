برمجة جدار ناري

تعلم كيفية إنشاء جدار حماية بسيط باستخدام لغة Python لحماية جهازك.

جدار الحماية الأساسي باستخدام Python

  1. قم بتثبيت Python على جهازك إذا لم يكن مثبتاً
  2. افتح محرر النصوص أو بيئة التطوير المفضلة لديك
  3. انسخ الكود التالي:
import socket
import sys
from datetime import datetime

# إعداد جدار الحماية البسيط
class SimpleFirewall:
    def __init__(self):
        self.allowed_ports = [80, 443]  # المنافذ المسموحة (HTTP, HTTPS)
        self.blocked_ips = []  # عناوين IP المحظورة
        
    def check_packet(self, ip, port):
        if ip in self.blocked_ips:
            return False
        if port not in self.allowed_ports:
            return False
        return True

# استخدام الجدار الناري
if __name__ == "__main__":
    firewall = SimpleFirewall()
    
    # إضافة عنوان IP إلى القائمة السوداء (مثال)
    firewall.blocked_ips.append("192.168.1.100")
    
    # محاكاة حركة مرور
    test_ip = "192.168.1.101"
    test_port = 80
    
    if firewall.check_packet(test_ip, test_port):
        print(f"السماح بالاتصال من {test_ip} عبر المنفذ {test_port}")
    else:
        print(f"حظر الاتصال من {test_ip} عبر المنفذ {test_port}")

هذا مثال بسيط لجدار حماية يمكنك تطويره بإضافة المزيد من الميزات مثل:

  • تسجيل محاولات الاتصال المشبوهة
  • كشف هجمات DDoS الأساسية
  • حظر نطاقات عناوين IP كاملة

نصائح للحماية من الاختراقات

10 نصائح أساسية لأمانك السيبراني

  1. استخدم كلمات مرور قوية وفريدة لكل حساب
  2. فعّل المصادقة الثنائية (2FA) حيثما أمكن
  3. حافظ على تحديث نظام التشغيل والتطبيقات
  4. تجنب استخدام شبكات Wi-Fi العامة للعمليات الحساسة
  5. احذر من رسائل البريد الإلكتروني والروابط المشبوهة
  6. استخدم برنامج مكافحة فيروسات موثوق به
  7. قم بعمل نسخ احتياطية منتظمة لبياناتك المهمة
  8. قلل من مشاركة المعلومات الشخصية على الإنترنت
  9. استخدم اتصال VPN عند التعامل مع بيانات حساسة
  10. تعلم أساسيات الأمن السيبراني لزيادة وعيك
